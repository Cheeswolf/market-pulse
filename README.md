# 📈 MarketPulse：AI金融市场情绪与新闻分析系统

## 🧠 项目简介

MarketPulse 是一个基于大语言模型（LLM）的金融新闻分析系统。
用户输入财经新闻或市场事件描述，系统自动输出结构化分析结果，包括摘要、市场情绪、影响资产、风险等级以及逻辑链条，用于辅助投资研究与信息整理。

---

## 🚀 核心功能

* 📰 **新闻摘要提取**：自动提炼核心信息
* 📊 **市场情绪判断**：输出偏多 / 偏空 / 中性
* 💰 **影响资产识别**：识别受影响的股票、债券、商品等
* ⚖️ **利多利空分析**：判断市场方向
* 🔗 **逻辑链构建**：生成“事件 → 变量 → 市场反应”推理路径
* 🗂 **历史记录功能**：支持分析结果存储与回溯

---

## 🧩 系统流程

```text
用户输入新闻
    ↓
Prompt 构建（结构化约束）
    ↓
大模型推理（LLM）
    ↓
结果解析（Parser）
    ↓
结构化 JSON 输出
    ↓
Streamlit 页面展示 + 本地存储
```

## ⭐ 项目亮点

### 1️⃣ 结构化输出设计

将非结构化财经文本转化为标准 JSON 输出，提高数据可用性与可扩展性。

### 2️⃣ Prompt 工程优化

通过严格约束模型输出格式（JSON），减少幻觉与解析失败问题。

### 3️⃣ 金融逻辑建模

构建“事件 → 变量 → 市场反应”的推理链条，例如：

> 通胀上升 → 利率预期上行 → 股市承压

### 4️⃣ 模块化架构设计

将系统拆分为 prompt / llm / parser / ui 模块，实现清晰的数据流与可维护性。
